# 变量提升 预处理

js引擎在代码正式执行之前会做一个预处理的工作：1、收集变量、2、收集函数

var 将var后面的变量定义，但是不赋值，

function(){} 提前定义函数

  1. JavaScript 中，函数及变量的声明都将被提升到函数的最顶部。
  2. JavaScript 中，变量可以在使用后声明，也就是变量可以先使用再声明。

# 初始化不会提升

JavaScript 只有声明的变量会提升，初始化的不会。

  1. 看两个例子： 内部函数可以使用外部变量；
var a=1;
function aa(){
    console.log(a);
    a++;
    console.log(a)
}
aa();
