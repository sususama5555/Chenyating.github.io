# vue-cli打包

# 前期准备

  1. 新建一个vue-cli项目：
vue init webpack 项目名
1
  1. 执行这个新项目：
npm run dev
1
  1. 打包这个新项目
npm run build
1

# 初次打包的坑

这时候你就会发现打包后的页面是一片空白的；

按F12在控制台上，会发现报错；file:///F:/static/css/app.30790115300ab27614ce176899523b62.css net::ERR_FILE_NOT_FOUND; 全是css，js路径引用错误的问题。

# 解决：到config文件夹中打开index.js文件。

更改build里面的assetsPublicPath属性： assetsPublicPath属性作用是指定编译发布的根目录，‘/'指的是项目的根目录 ，'./'指的是当前目录。

修改以后，打包，页面就有内容了。

# 我们需要安装cross-env

npm install --save-dev cross-env
1

先修改script脚本,设置了两个环境变量 NODE_ENV 和 env_config然后执行正常的打包

  "scripts": {
    "dev": "webpack-dev-server --inline --progress --config build/webpack.dev.conf.js",
    "start": "npm run dev",
    "build": "node build/build.js",
    // 添加两条新命令
    "build:test": "cross-env NODE_ENV=production env_config=test node build/build.js", 
    "build:prod": "cross-env NODE_ENV=production env_config=prod node build/build.js" 
  }
