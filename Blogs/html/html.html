# doctype

<!DOCTYPE html>：<!DOCTYPE>声明有助于浏览器中正确显示网页。

网络上有很多不同的文件，如果能够正确声明HTML的版本，浏览器就能正确显示网页内容。

doctype 声明是不区分大小写

# 定位机制

  • 普通流
  • 浮动流
  • 绝对定位

html网页就是一个标准文档流；

性质：

  • html文本之间有空白折叠现象；
  • 文字，图片，表单元素，这些文本类型的元素有一个特点，高矮不齐，底边对齐；
  • 文字在盒子内如果一行书写完毕，会自动换行；

# 元素划分

块级标签：所有的容器级都是块级标签，文本级标签的p也是块级标签； 行内标签：出了p意外所有文本级标签

# 块级元素

body、div、h1-h6，p,ul,ol,li,dl,dt,dd,table,tr,td等

特点：

  • 块级元素会独占一行，同行不会出现其他同级标签；
  • 可以设置宽高；
  • 嵌套的关系，子盒子不设置宽度，继承父盒子的宽度；

# 行内元素

a,span,img,input等文本类

特点：

  • 同行出现其他同级标签；
  • 行内元素设置宽高无用；内边距和外边距可以设置；
  • 行内元素不设置宽度，不会继承父亲宽度；只能用内容撑开；

# 行内块标签

既可以设置宽高，还可以在一行排列；

互相转换的方法：

# history对象

包含浏览器的历史url的集合

api：

  • history.back() 与浏览器点击后退按钮相同;
  • history.forward() 与浏览器点击向前按钮相同；
  • history.go() 进入历史中的某个页面；

# location对象

用于获得当前页面的地址(url),并把浏览器重定向到新的页面；

属性：

  • location.hostname：返回web主机的域名
  • location.pathname； 返回当前页面的路径和文件名
  • location.port:页面端口
  • location.protocol：页面协议
  • location.href：页面路径
  • location.assign();加载新的文档
